## Frozen-Simulate
Instagramのアカウント作成時に稀に起こる**凍結**を体験できる[サイト](https://pop-apple.github.io/Frozen-Simulate/)です  

## 遊び方
画面上の緑枠内のユーザーネームとパスワードをテキストボックスに入力して  
ログインボタンをクリックして下さい。  
画面が切り替わりあなたのアカウントが凍結したとの通知が来ます。
